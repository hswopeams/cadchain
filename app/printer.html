3DID Printer Page

The contract owner can use the Admin Page to register printers and designers.

Designers can use the Designer Page to register designs. and withdraw funds.

To Register or access a design, a hash that is assumed to be a pointer to the design on IPFS or other file system must be provided. Some sample values are:

0x56656e74696c61746f72506f696e746572000000000000000000000000000000
0x56616c766532506f696e74657200000000000000000000000000000000000000
0x56616c7665506f696e7465720a00000000000000000000000000000000000000

This site can be used to generate more hashes
